WellPivot LP

WellPivot LP

Программный комплекс «WellPivot LP – Анализ данных» (ПК WellPivot LP) позволяет пользователям проводить детальный анализ данных, просматривать инфографику, получать выгрузки данных в различных форматах.

WellPivot LP может применяться специалистами нефтегазодобывающих и буровых предприятий нефтегазовой отрасли.

Назначение

ПК WellPivot LP работает на отечественной операционной системе Astra Linux (Орел или Смоленск). В качестве базы данных используется Postgre SQL 11.5 или новее.

Основная цель использования программного комплекса – автоматизация процесса составления всех видов отчетности с производственных объектов, выполнение технико-экономического анализа деятельности предприятия и оценки качества выполняемых работ и исследований.

Возможности

Источниками информации для выполнения аналитики являются следующие данные:

  • проектная документация на строительство скважины;
  • ежесуточная отчетность сервисных подрядчиков, формируемая в процессе строительства скважины;
  • автоматизированные данные ГТИ, MWD\LWD;
  • исходные финансовые показатели при строительстве скважин.

Применение ПК WellPivot LP предоставляет пользователю возможность детально анализировать данные по любым объектам, включая ДО, месторождения, скважины, секции, рейсы и т.д.; проводить детальный анализ работы подрядчиков, анализ НПВ на скважине; «дела скважины»; а также просматривать плановые и фактические графики бурения.

Особенности

ПК WellPivot LP имеет широкий спектр применения, от учета и построения отчетов по одному объекту бурения, до проведения анализа данных по целому предприятию.

Основные инструменты для проведения комплексного анализа:

  1. Отображение оперативной информации по текущему состоянию объектов бурения, группировка и выбор объектов для дальнейшего анализа с помощью фильтров по различным критериям (Филиал, Месторождение, Куст, Скважина, Назначение скважины, Профиль скважины, Тип скважины, Количество дополнительных стволов, Тип БУ и другие.).

  2. Просмотр детальной информации по выбранным объектам.

  3. Получение динамики ключевых показателей эффективности. ПК WellPivot LP позволяет проследить динамику ключевых показателей эффективности по заданным критериям:

    • Аварийность: Количество аварий, Количество осложнений, Коэффициент аварийности, Коэффициент тяжести аварий.
    • Данные по перебурам по геологическим или технологическим причинам: Кол-во перебуров, НПВ из-за перебуров, сут., НПВ из-за перебуров, %, Проходка по перебурам, Продолжительность перебуров.
    • Данные по продолжительностям: Бурения, НПВ, сут., НПВ, %, ПВ, сут., Общее время план, сут., Общее время факт, сут.
    • Данные по проходке: Скорость бурения, Коммерческая скорость, Проходка общая, Проходка в пилотном стволе, Проходка по перебурам, Проходка эффективная, Глубина, план, Глубина, факт, Глубина пилотного ствола, Длина горизонтального участка.
    • Данные по типам НПВ: Авария, сут., Авария, %, Брак, сут., Брак, %, Ремонт, сут., Ремонт, %, Осложнение, сут., Осложнение, %, Осложнение по геологическим причинам, сут., Осложнение по геологическим причинам, %, Простой по метеоусловиям, сут., Простой по метеоусловиям, %, Простой, сут., Простой, %.

    • Прочие показатели: Кол-во бригад бурения, Кол-во скважин, Кол-во законченных бурением.

  4. Диаграмма Ганта. На диаграмме отображаются плановый и фактический графики бурения скважин с возможностью визуализации продолжительности бурения, % НПВ, сравнения плановых и фактических продолжительностей бурения скважин.

  5. «Дело скважины». Просмотр всех документов по скважине, имеющихся в базе данных. Построение сводных отчетов по выбранным объектам с заданными характеристиками в формате xlsx таких, как Суточный рапорт, Отчет по балансам времени, Сводный отчет по НПВ по подрядчикам, Анализ работы долот и др.

  6. Автоматизированная рассылка отчетов по электронной почте. Комплекс позволяет выполнять автоматическую рассылку требуемых отчетов специалистам компании к указанному времени.
  7. Каталогизация и хранение документации по объектам строительства.
  8. Контроль данных. Комплекс позволяет проводить анализ качества и полноты данных ГТИ и производственной отчётности, получаемых с буровой площадки.

Использование ПК WellPivot LP позволяет многократно ускорить формирование сводных отчетов и проведение анализа производственных показателей. ПК WellPivot LP интегрирован с информационной системой «Удаленный мониторинг бурения LP» (ИС УМБ LP).
